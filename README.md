# Les Joies de la Wild project #

A [Wild Code School's project](https://wildcodeschool.fr/)

## Dependancies ##
**Bootstrap** : 3.3.7  
**JQuery** : 3.3.1

## Directory Structure of the project ##  
* **root directory**  
    * **css** : CSS files  
    * **js** :  Javascript files  
    * **src** : PHP files
    * **vendor** : library 
        * **bootstrap**  
        * **jquery**
    * **assets** : internal ressources
        * **fonts** : fonts directory  
        * **img** : images directory
